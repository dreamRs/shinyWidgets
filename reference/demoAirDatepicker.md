# Some examples on how to use airDatepickerInput

Some examples on how to use airDatepickerInput

## Usage

``` r
demoAirDatepicker(example = "datepicker")
```

## Arguments

- example:

  Name of the example : `"datepicker"`, `"timepicker"`, `"months"`,
  `"years"`, `"update"`.

## Examples

``` r
if (interactive()) {

  demoAirDatepicker("datepicker")

}
```
