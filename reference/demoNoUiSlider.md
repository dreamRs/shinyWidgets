# Some examples on how to use noUiSliderInput

Some examples on how to use noUiSliderInput

## Usage

``` r
demoNoUiSlider(example = "color")
```

## Arguments

- example:

  Name of the example : `"color"`, `"update"`, `"behaviour"`, `"more"`,
  `"format"`.

## Examples

``` r
if (interactive()) {
```
